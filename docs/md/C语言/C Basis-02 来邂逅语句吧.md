### Step 2 聊聊函数吧

> 我们在写代码实现一些功能时，难免会有一些代码重复的被使用，这时候我们便可以将这段代码整理起来，将其封装成为能够独立完成某个功能的函数。
>
> 接下来我们来简单了解一下什么是函数。

阅读以下程序，并回答问题

```C
int sum(){
    int i, sum=0;
    for(i=1; i<=100; i++){
        sum+=i;
    }
    return sum;
}
```

2. 描述函数实现什么功能
3. 描述形式参数与实际参数的区别
4. 尝试用函数包装 step1中两数相加的功能
